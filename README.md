# CS F429 (3 0 3): Natural Language Processing (Semester II 2025-26)

This repository contains lecture-aligned live demos, hands-on notebooks, and reference material for the course **CS F429 (3 0 3) – Natural Language Processing**, offered at **BITS Pilani, Dubai Campus**.

## About This Repository

The goal of this repository is to support **concept-first learning** in NLP by combining:

- Interactive, live-coded examples from lectures  
- Minimal, reproducible preprocessing and modeling pipelines  
- Clear demonstrations of how real-world text differs from textbook data  

## Scope of the Materials

The materials progress from **foundational NLP concepts** such as:
- Tokenization  
- Sentence segmentation  
- Feature extraction  

to **modern neural NLP approaches**, including:
- Word and sentence embeddings  
- Transformer-based models  

Emphasis is placed on understanding **why each step exists**, not just how to use a library.

## How Students Should Use This Repository

Students are encouraged to:
- Run notebooks during and after lectures  
- Modify examples to observe model behavior and failure cases  
- Use this repository as a reference for assignments and project work  

## Updates

This repository will be updated incrementally across the semester, aligned with lecture topics.
